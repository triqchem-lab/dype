Description

Dayan.Compute.CRT — 中国剩余定理 (CRT) 全局查表

前向: 6624 个格点索引 → (极向模 144, 环向模 46) 逆向: CRT 重构 (Bézout 系数法)

公式: 前向: f(i) = (i % 144, i % 46) 逆向: g(p, t) = p · (46 · inv46_144) + t · (144 · inv144_46) mod 6624 其中 inv46_144 = 46⁻¹ mod 144, inv144_46 = 144⁻¹ mod 46

复杂度: O(1) 查表, O(6624) 一次性构建

对齐 Agda: Sovereign.Arithmetic.CRTLemmas.agda CRT 模互质 (gcd(144,46)=2, 故 6624=LCM)
