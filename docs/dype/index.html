dype

The Da-Yan Proof Engine (DY-PE) is a discrete lattice computation engine that generates Agda proof terms. It replaces Agda's λ-calculus reduction with CRT table lookups, orbit decomposition, and limit-cascade phase alignment on the T⁶ torus lattice (729 points).
