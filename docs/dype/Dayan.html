decode :: Tryte -> [Trit] Source #

将 Tryte 解码为 6 个 Trit 列表 [v0, v1, v2, v3, v4, v5]

算法: 6 层 %3 + /3 剥离 (4320D 范式) v0 = n % 3, n₁ = n / 3 v1 = n₁ % 3, n₂ = n₁ / 3 ... v5 = n₅ % 3

encode :: [Trit] -> Maybe Tryte Source #

encode [v0, v1, v2, v3, v4, v5]

其中 vi 的 ℕ 编码为 N→0, Z→1, P→2

newtype Tryte Source #

Tryte — 6-trit 格点, 编码为 0..728 的 Word16

内部编码: 小端基3 (v0 在 LSB) index = v0 + 3*v1 + 9*v2 + 27*v3 + 81*v4 + 243*v5

不变式: 0 ≤ index < 729
