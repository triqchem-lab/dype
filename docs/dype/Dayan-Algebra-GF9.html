Description

Dayan.Algebra.GF9 — GF(3²) 的 CRT 余数向量表示

幻方正交拓扑基石: i² + 1² = 0² M4 正交本征方向: v16⁺ (右手征) ⊥ v16⁻ (左手征) crtLabel e16⁺ = crtLabel e16⁻ = 16 (CRT投影相同!) 差异在 C3 层: crt-to-trit e16⁺ = T₁ (右旋), e16⁻ = T₂ (左旋) eigen-conjugate: e16⁺ ↔ e16⁻

GF(9) 的 9 个元素 → CRT 余数向量: 实部 a ∈ {0,1,2} → CRT 投影 (a % 144, a % 46) 虚部 b ∈ {0,1,2} → C3 手征: 0=中性, 1=右旋(T₁), 2=左旋(T₂) Frobenius σ(a,b)=(a,-b): 手征翻转, CRT投影不变 共轭对 (a,1)↔(a,2) 共享同一 CRT 坐标

对齐 Agda: AlgebraGF9.agda, QuantumBridge §24 (crtLabelcrt-to-trit)
