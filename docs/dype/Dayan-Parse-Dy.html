Dayan.Parse.Dy

Description

Dayan.Parse.Dy — .dy 语法解析器 (基于 Agda 语法子集)

支持完整 .dy 语法: pragma, module, open import, postulate, data, rewrite, defs 类型: Set, Nat, Fin n, Vec A n, A -> B, (x : A) -> B 项: refl, 字面量, 变量, 应用, {!!} 运算符: _===_, _+_, _*_, _%_, _/_

不支持的语法会报 ParseError, 而非静默跳过。
