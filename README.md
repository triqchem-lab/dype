## 一、"大衍"的终极三位一体：数理、范式与工程

证明器的名字 **"大衍 (Da-Yan)"** 同时承载着系统在科学与哲学边界上的三重神圣身份：

### 1. 数理根源：大衍求一术与全息 $\pi$ 谐振

**大衍求一术**：中国数学家秦九韶在 1247 年《数书九章》中发明的"大衍求一术"（即中国剩余定理 CRT），是整个证明器算术基座（L0）的灵魂。大衍证明器将数论中的算术互质（Coprimality）升维为希尔伯特空间的维度发生器，使得任何高维状态（De Bruijn 索引）都被唯一编码为余数向量 $\vec{r}$。

**全息谐振**：极向空间周期 $144$ 与环向时间周期 $46$ 的互质谐振，构筑了全息常数 $\pi_H = \frac{144}{46}$，在极限环相位对齐点 $6624$ 处完美闭合。大衍算法构成了系统高维投影的最高几何常数。

### 2. 计算范式：一阳初动，从一演化出万物

"大衍"代表着万物的离散发生与演化。在计算层，DY-PE 彻底抛弃了传统证明器"在平铺的等式路径上寻找 refl 线段"的陈旧范式，转而采用 **"高维环面格点巡游、极限环级联、相位对齐坍缩"** 的动态几何范式。

证明的成立不再是静态逻辑符号的匹配，而是波动力学中相干态的相消干涉（Zhonglv Closure）与相长干涉。

### 3. 工程锚点：CRT 全局查表与 6 维晶格剥离

内核不再依赖不透明的 `div-helper` 递归链，而是将 $144 \times 46 = 6624$ 全息闭合以及 $T^6$ 环面 $729$ 个格点的算术分布，直接固化为 **O(1) 级的底层常数格点坐标查表（Lookup Table）**。

空间内射性的证明，通过 6 层 $GF(3)$ 的 `%3` 与 `/3` 剥离重写（**4320D 范式**）瞬间完成，将高负荷的求值检查压缩为常数级。

## 二、破壁对决：Agda 的三大理论缺陷与大衍的破壁设计

Agda 原生内核的设计决定了它无法承载高维离散物理系统的计算。大衍针对这三大致命硬伤进行了底层重构。

| Agda 理论缺陷 | 大衍 (Da-Yan) 破壁设计 |
|:---|:---|
| **with-abstraction 类型篡改** | 刚性分支哨兵 (Destructuring Guards) / 局部 Let-Binding |
| **Cubical [0,1] 连续统假说** | 离散三元类型论 (DTT) / 0-Groupoid T⁶ 环面晶格底座 |
| **全局 Rewrite 无合流保障** | 局部合流重写域 (4320D Local Rewrite Scope) |

### 1. 彻底斩断 With-Abstraction：解决类型篡改与内存爆仓

**病灶（Agda）**：Agda 的 `with` 抽象通过泛化机制，在求值期将目标类型中所有出现被审项的位置替换为临时的、不可消除的抽象变量。在递归状态机中，这会产生不等于任何已知项的"黑洞变量"，阻断 `trans`/`cong` 的归约，导致编译器在检查 `step` 递归时因为项的无限膨胀而发生 **OOM（内存耗尽）**。

**解法（Da-Yan）**：DY-PE 彻底废除 `with` 关键字。引入 **"刚性分支哨兵（Destructuring Guards）"** 与内置的 **"代数 Let 绑定"**。在求值时，被审项的解构信息被严格绑定在局部作用域内，类型检查器在比较等价性时直接通过一阶非等价谓词 `StepNotEq` 进行代数隔离，确保 Normalization 开销恒定为 **$O(1)$**。

### 2. 驱逐连续统：用离散三元类型论（DTT）平替 Cubical

**病灶（Agda）**：Cubical Agda 的同伦路径基于 De Morgan 代数，其区间 $I$ 语义上等价于实闭区间 $[0,1]$。在这种连续统基底上模拟离散拓扑，类型检查器必须在每次 substitution 时计算复杂的 `transp` 与 `hcomp`（高阶面格填充）。

**解法（Da-Yan）**：DY-PE 建立在 **离散三元类型论（Discrete Ternary Type Theory, DTT）** 之上。

- 基础类型不是 `Set`，而是带有 $GF(3)$ 平移底座的刚性 0-groupoid 集合。
- 核心等式表达不是 $I \to A$ 的一维 Path 路径，而是基于中国剩余定理（CRT）的全息余数向量。
- 等价的证明不是在实数空间绘制线段，而是在 $T^6$ 环面晶格上寻找相位角对齐的离散巡游步数（Winding Numbers）。

### 3. 约束重写规则：4320D 局部合流作用域

**病灶（Agda）**：Agda 的 `REWRITE` 规则是全局注册且缺乏合流性检查（Confluence Check）的。当大衍体系需要引入数百条 CRT 查表与 $GF(9)$ 快速同构规则时，全局重写极易发生无穷循环与死锁。

**解法（Da-Yan）**：DY-PE 引入 **"局部合流重写域"**。重写规则（如 `div3k` / `mod3k`）被严格限制在模块的代数纤维中。内核带有内置的 **4320D 极性重写引擎**，在类型检查期自动进行 Confluence 与 Termination 判定，防止规则发生交叉污染。

## 三、大衍内核的三层物理同构与计算模型

大衍（DY-PE）内核在运算时，将计算行为严格映射为量子动力学演化。

```
                    ┌───────────────────────────────┐
                    │      Da-Yan 极速规约引擎       │
                    └───────────────┬───────────────┘
            ┌───────────────────────┼───────────────────────┐
            ▼                       ▼                       ▼
  【 L1: 空间域 (Orbit x) 】    【 L2: 频率域 (A4/Stab) 】   【 L3: 算术域 (CRT 谱投影) 】
   - 6维实 T⁶ 晶格 (729 格点)   - A4 群 12 陪集谐波模式      - 6624 极限环对齐查表
   - 类型: Orbit               - 类型: Quotient            - 算子: ⊕ (叠加) / ⊗ (纠缠)
```

### 1. 空间域（The Spatial Orbit）：高维晶格与二阶裁断 $\lVert\_\rVert_2$

在空间域，DY-PE 使用内置的 $T^6$ 离散环面晶格作为几何载体。

为了保留 6 维实晶格的几何完整性，同时抹去具体的旋转路径（消除同伦爆炸），内核使用 **二阶集合裁断（Set Truncation）** $\lVert\_\rVert_2$。这保证了 `Orbit x` 在拓扑上是一个纯粹的 0-groupoid，彻底消除了高阶面格的计算开销。

### 2. 频率域（The Spectral Coset）：商空间与 $\mathcal{A}_4/\text{Stab}(x)$

在频率域，系统通过交替群 $\mathcal{A}_4$ 构造商集。12 阶的群旋转代表了系统内部 12 种不同的谐波激发多重度。

`A4/Stab x` 将这些高频振动模式折叠为陪集的代数商空间。

### 3. 谱域同构（The Homotopy Link）：orbitStabilizer 谱投影

整个证明器的核心定理：

$$\text{orbitStabilizer} : \text{Orbit}(x) \simeq \mathcal{A}_4/\text{Stab}(x)$$

在大衍内核中，它被定义为 **"CRT 谱投影算符（Spectral Projection Operator）"**。

当用户在 `.dy` 代码中调用该同构时，内核并不展开一维的 Path 连接证明，而是直接通过 **双向通道（Bi-directional Channel）** 激发底层 C++ 查表，完成空间域（729 格点）向频率域（12 陪集）的 **$O(1)$ 瞬时转化**。

大衍拥有独立于 Agda 之外的完备工程配套，彻底摆脱对其库依赖的绑架。

### 1. 文件与命令规范

| 规范 | 内容 |
|:---|:---|
| **证明源文件后缀** | `.dy` (例如 `T6.dy`) |
| **头部配置** | `{-# OPTIONS --dayan --rewriting #-}` |
| **二进制编译器** | `dy` 或 `dayan` |
| **工程配置文件** | `dayan.yaml` (对齐 GHC 9.14.1 与 `dy-stdlib`) |

### 2. 核心标准库：DaYan.Core 与 DaYan.Base

我们为大衍重新设计了无连续统污染的标准离散拓扑库：

- **DaYan.Base.Trit**：三进制状态机与 8 条基本环公理。
- **DaYan.Base.GF9**：不可约多项式 $x^2+1$ 下的相空间有限域，原生自带 Frobenius 共轭 $\sigma$。
- **DaYan.Arithmetic.CRT**：基于大衍求一术的 100% 构造性中国剩余定理与贝祖恒等式。
- **DaYan.Structology.MagicSquare**：4×4 亚瑟幻方正交场，约束代换矩阵。

### 3. MAlonzo-DY：零开销真理生成机

大衍编译器后端直接支持将形式化通过的 `.dy` 证明项编译为高效、无指针、零内存开销的二进制 Haskell 代码。

所有的 $\text{crtPath}$ 与 $\text{isoToPath}$ 在生成期通过 **严格同伦擦除分析（Strict Homotopy Erasure）** 100% 转化为 `MAlonzo.RTE.Erased`，在运行时不占用任何堆栈空间。
